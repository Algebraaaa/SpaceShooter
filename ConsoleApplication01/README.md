# Space Shooter - OpenGL 太空射击游戏

一个使用 OpenGL 和 GLUT 开发的 2D 太空射击游戏。

## 游戏简介

玩家控制一艘飞船，抵御一波又一波的敌人进攻。击败敌人获得分数，尽可能生存更长时间！

## 游戏特性

- ?? 玩家飞船控制与射击系统
- ?? 三种敌人类型：
  - **普通敌人**（红色）- 标准速度，1点生命值
  - **快速敌人**（黄色）- 高速移动，带翅膀
  - **坦克敌人**（灰色）- 移动缓慢，3点生命值
- ?? 粒子爆炸效果
- ? 星空背景
- ?? 波数递进系统，难度逐渐增加
- ?? 完整的游戏状态管理（菜单、游戏、暂停、结束）

## 操作说明

| 按键 | 功能 |
|------|------|
| W | 向上移动 |
| S | 向下移动 |
| A | 向左移动 |
| D | 向右移动 |
| 空格 | 射击 |
| 鼠标左键 | 射击 |
| P | 暂停/继续 |
| Enter | 开始游戏/重新开始 |
| ESC | 退出游戏 |

## 编译要求

- C++14 或更高版本
- OpenGL
- GLUT (freeglut)
